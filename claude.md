# Kos Putri Semarang Indah

Website untuk kos-kosan khusus putri yang berlokasi di Semarang Indah, Semarang.

## Deskripsi Project

Website informasi dan booking untuk kos putri dengan fitur:
- Informasi kamar dan fasilitas
- Galeri foto kos
- Sistem booking/reservasi
- Kontak dan lokasi
- Testimoni penghuni

## Target Pengguna

- Mahasiswi yang mencari kos di area Semarang Indah
- Karyawati yang bekerja di sekitar Semarang
- Orang tua yang mencari kos aman untuk anak putri mereka

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (atau React/Next.js)
- **Styling**: Tailwind CSS
- **Backend**: Node.js/Express atau PHP (opsional)
- **Database**: MySQL/PostgreSQL (jika diperlukan sistem booking)

## Informasi Kos

### Alamat
Semarang Indah, Kota Semarang, Jawa Tengah

### Fasilitas Umum
- WiFi gratis
- Parkir motor
- Dapur bersama
- Ruang tamu
- Kamar mandi dalam/luar
- Keamanan 24 jam
- CCTV
- Laundry (opsional)

### Tipe Kamar
1. **Kamar Standar** - Kamar mandi luar, kasur, lemari, meja belajar
2. **Kamar Premium** - Kamar mandi dalam, AC, kasur, lemari, meja belajar

### Peraturan Kos
- Khusus putri
- Jam malam (maksimal jam 22.00)
- Tamu lawan jenis diterima di ruang tamu
- Dilarang membawa hewan peliharaan
- Menjaga kebersihan dan ketenangan
